Workshop 9: Arrays with JavaScript

Workshop 9

An Array is a special variable, which can hold more than one value at a time. Arrays allow you to store multiple values in a single variable, making it easy to work with lists of data.

Creating Arrays

You can create an array using square brackets [] and placing values inside them, separated by commas:


const colours = ["Red", "Green", "Blue"];

In this example, colours is an array containing three strings.

Accessing Array Elements

You can access an element in an array using its index, starting from 0:


console.log(colours[0]);  // Outputs: Red
console.log(colours[1]);  // Outputs: Green
console.log(colours[2]);  // Outputs: Blue

Modifying Arrays

You can change an array element by accessing it with its index and assigning it a new value:


colours[1] = "Purple"; // Changes "Green" to "Purple"
console.log(colours);   // Outputs: ["Red", "Purple", "Blue"]

Array Properties and Methods

Arrays come with several built-in properties and methods. Here are some commonly used ones:

1. Length

The length property returns the number of elements in an array:


console.log(colours.length);  // Outputs: 3

2. Push and Pop

push() adds a new element to the end of the array, while pop() removes the last element:


colours.push("Yellow"); // Adds "Yellow" to the end
console.log(colours);    // Outputs: ["Red", "Green", "Blue", "Yellow"]

colours.pop();           // Removes the last element
console.log(colours);    // Outputs: ["Red", "Green", "Blue"]

3. Shift and Unshift

shift() removes the first element, while unshift() adds a new element at the beginning of the array:


colours.shift();          // Removes the first element
console.log(colours);     // Outputs: ["Green", "Blue"]

colours.unshift("Black"); // Adds "Black" to the beginning
console.log(colours);     // Outputs: ["Black", "Green", "Blue"]

Common Array Methods

Here are a few additional array methods that are useful for manipulating data:

  • concat() - Joins two or more arrays.
  • indexOf() - Returns the first index of a specified element.
  • includes() - Checks if an array contains a specified element.
  • slice() - Returns a portion of an array.
  • reverse() - Reverses the order of the array.

Example: Using `concat` and `slice`


const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const combined = arr1.concat(arr2);  // Outputs: [1, 2, 3, 4, 5, 6]

const sliced = combined.slice(2, 5); // Outputs: [3, 4, 5]
